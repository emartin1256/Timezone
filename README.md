## Overview

This is a live timezone app. Users can search for timezone information using the searchbar and results show dynamically when typing.

## Setup

In the command line:
```bash
git clone
```
```bash
cd Timezone
```
```bash
cd api
```
```bash
node app.js
```
```bash
cd ..
```
```bash
cd timezones
```
```bash
npm start
```
